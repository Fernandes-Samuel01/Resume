# 📄 Personal Resume Webpage (HTML Only)

## 📌 Project Overview

This project is a **single-page personal resume website** built using **pure HTML**.
The main goal of this assignment is to demonstrate the use of **semantic HTML elements**, including tables, lists, and forms — without using CSS.

---

## 🎯 Objectives

* Structure a resume using semantic HTML tags
* Display data using tables
* Organize information using lists
* Collect user input through forms

---

## 🛠️ Technologies Used

* HTML5

---

## 📂 Features

### 🔹 1. Basic Profile Section

* Name displayed using `<h1>`
* Professional summary using `<p>`
* Profile image using `<img>` with alt text

---

### 🔹 2. Education Section (Table)

* Structured using `<table>`
* Includes:

  * Year
  * Degree/Course
  * Institution
* Uses `<thead>` and `<tbody>` properly

---

### 🔹 3. Experience & Skills

* **Projects/Experience** → Unordered List (`<ul>`)
* **Technical Skills** → Ordered List (`<ol>`)

---

### 🔹 4. Contact Form ("Hire Me")

Includes:

* Name input
* Email input
* Dropdown (Reason for contact)
* Message textarea
* Submit button
